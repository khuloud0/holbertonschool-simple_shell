# Simple Shell

## Description
Simple Shell is a simple command-line interpreter in C, inspired by the Unix shell.

## Installation
git clone https://github.com/khuloud0/holbertonschool-simple_shell.git
cd holbertonschool-simple_shell
gcc -Wall -Werror -Wextra -pedantic *.c -o hsh

## Usage
./hsh
